## 文件名生成规则

### 日期格式
使用 ISO 8601 标准日期格式: `YYYY-MM-DD`

### 任务类型标识
- 重构任务: `重构`
- 新需求任务: `新需求`

### 模块名称规范
- 使用中文或英文，保持统一
- 避免使用特殊字符
- 长度控制在 20 个字符以内

### 完整示例
```
2025-05-14_重构_用户中心页面.md
2025-05-14_新需求_订单导出功能.md
2025-05-14_重构_支付接口优化.md
2025-05-14_新需求_消息推送系统.md
```

## 飞书文档标题格式

### 格式规则
```
[{任务类型}] {模块/页面名称} - {日期}
```

### 示例
```
[重构] 用户中心页面 - 2025-05-14
[新需求] 订单导出功能 - 2025-05-14
```
